# Text Formatting in HTML

HTML contains several elements for defining text with a special meaning:

- `<b>` - Bold text
- `<strong>` - Important text
- `<i>` - Italic text
- `<em>` - Emphasized text
- `<mark>` - Marked text
- `<small>` - Smaller text
- `<sub>` - Subscript text
- `<sup>` - Superscript text
- `<del>` - Deleted text
- `<ins>` - Inserted text
